# MiniLibrary Project

This project is a simple library system implemented in Java using Object-Oriented Programming (OOP) principles.

## Features
- Add books to the library
- Manage readers
- Borrow and return books
- Track book availability

## Classes
- Book → represents a book
- Reader → represents a user
- Library → manages books and operations
- Main → test scenario

## Scenario
The main method demonstrates:
- adding books
- printing available books
- borrowing a book
- returning a book

## Technologies
- Java
- Git & GitHub
